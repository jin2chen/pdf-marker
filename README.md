# PDF marker tool

[Demo](https://jin2chen.github.io/pdf-marker/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```


### Publish
```
npx gh-pages -d dist -f
```
